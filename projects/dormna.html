DormnetAI – Roommate Matching System

DormnetAI is a full-stack web application designed to match university dorm roommates based on lifestyle, academic, and behavioral preferences.

Problem

Random roommate assignments often lead to conflicts and poor living experiences due to incompatible habits and expectations.

Solution

Implemented a questionnaire-driven system combined with data normalization and machine-learning-based similarity scoring to generate optimal roommate matches.

Architecture & Tech

  • Frontend: React
  • Backend: Node.js, Express
  • Database: PostgreSQL
  • Data Processing: Python, pandas
  • ML: scikit-learn similarity scoring
