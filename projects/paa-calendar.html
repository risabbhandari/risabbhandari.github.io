Paa Calendar Project

A digital calendar system integrating Nepali Bikram Sambat and Newari lunar dates into standard calendar formats.

Problem

Regional calendars are often incompatible with mainstream digital calendar platforms such as Google and Apple Calendar.

Solution

Built a backend service that dynamically generates downloadable calendar files compatible with standard calendar clients.

Architecture & Tech

  • Backend: Python, FastAPI
  • Calendar Generation: icalendar
  • APIs: Google Calendar API
  • Frontend: Next.js
