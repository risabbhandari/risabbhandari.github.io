Paakyo — Food Delivery Platform

A cross-platform mobile app connecting customers with home-based cooks and local restaurants. The backend provides APIs for orders, carts, and delivery workflows.

React Native Django REST PostgreSQL Firebase Google Maps API

What I built

  • Role-based flows for customers, cooks/restaurants, and delivery.
  • Backend order lifecycle + cart logic APIs.
  • Auth + notifications using Firebase Auth/FCM.
  • Maps + location features for delivery accuracy.
